## 1. Max Chunks to Make Sorted

### 🔹 Problem Understanding

We need to split an array into maximum number of chunks such that:

* Sorting each chunk individually
* Concatenating them
  👉 Results in a fully sorted array

---

### 🔹 Key Idea

At any index `i`, if the **maximum element till that index equals i**,
then all elements before it are in correct position → we can form a chunk.

---

### 🔹 Code

```java
public int maxChunksToSorted(int[] arr) {
    int max = 0, chunks = 0;

    for (int i = 0; i < arr.length; i++) {
        max = Math.max(max, arr[i]);

        if (max == i) {
            chunks++;
        }
    }
    return chunks;
}
```

---

### 🔹 Complexity

* Time → O(n)
* Space → O(1)

## 2. Group Strings by Frequency & Find Unique

### 🔹 Problem

Given a list of strings:

* Count frequency
* Return only unique elements

---

### 🔹 Approach

Use `groupingBy + counting`

---

### 🔹 Code

```java
Map<String, Long> freq = list.stream()
    .collect(Collectors.groupingBy(
        Function.identity(),
        Collectors.counting()
    ));

List<String> unique = freq.entrySet().stream()
    .filter(entry -> entry.getValue() == 1)
    .map(Map.Entry::getKey)
    .toList();
```

---

### 🔹 Interview Tip

* Return type → `Map<String, Long>`
* Use case → analytics, logs, duplicate detection

## 3. Two Threads Print 1–100 (Odd/Even)

### 🔹 Concept

We use:

* `wait()` → pause thread
* `notify()` → wake other thread
* `synchronized` → ensure mutual exclusion

---

### 🔹 Code

```java
class NumberPrinter {
    private int num = 1;

    public synchronized void printOdd() throws InterruptedException {
        while (num <= 100) {
            if (num % 2 == 0) {
                wait();
            } else {
                System.out.println("Odd: " + num++);
                notify();
            }
        }
    }

    public synchronized void printEven() throws InterruptedException {
        while (num <= 100) {
            if (num % 2 != 0) {
                wait();
            } else {
                System.out.println("Even: " + num++);
                notify();
            }
        }
    }
}
```

---

### 🔹 Key Points

* Avoid race conditions
* Thread coordination
